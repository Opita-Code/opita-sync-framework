# Repo boundary

Este repo contiene **OSF** como framework/kernel reusable.

Incluye:

- código Go del framework
- baseline reusable v1
- specs fase A-E
- implementation plan
- convergence profile
- definitions y demo de referencia

No incluye la capa de producto como source of truth principal.

La capa de producto vive en el repo hermano:

- **Opita Sync**

Regla dura:

- **OSF** conserva autoridad sobre runtime, policy, approvals, evidence y execution semantics.
- **Opita Sync** conserva autoridad sobre UX, tenant onboarding, catálogo visible, surfaces y roadmap de producto.
